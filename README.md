# Eagle Home Project

A modern, professional home services marketplace built with HTML, CSS, and JavaScript. Styled after Bark.com with a premium gold eagle branding theme.

## Features

✨ **Modern Design**
- Premium gold eagle logo (#D4AF37)
- Clean, professional interface
- Responsive layout for all devices
- Smooth animations and transitions

👥 **For Clients**
- Search and filter professional services
- View rated professionals with reviews
- Get quotes from multiple professionals
- Post projects for free
- Rate and review completed work

💼 **For Professionals**
- One-time $29 membership fee
- Zero commissions on jobs
- Unlimited job leads
- Build your professional reputation
- Direct client connections

📱 **Service Categories**
- Plumbing
- Electrical
- Carpentry
- Painting
- HVAC
- Cleaning
- Landscaping
- General Repair

## Getting Started

### Installation

1. Clone the repository:
```bash
git clone https://github.com/tradeapex67-eng/eagle-AI.git
cd eagle-AI
```

2. Open in your browser:
```bash
# Using a local server (recommended)
python -m http.server 8000
# Or with Node.js
npx http-server

# Or simply open index.html in your browser
open index.html
```

## Customization

### Colors
Edit the CSS variables in `styles.css`:
- Gold: `#D4AF37`
- Light Gold: `#FFD700`
- Dark: `#1A1A1A`

### Fonts
Using 'Inter' font family from Google Fonts. Change in `<head>`:
```html
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700;900&display=swap">
```

### Content
Edit text directly in `index.html`. All content sections are clearly marked.
